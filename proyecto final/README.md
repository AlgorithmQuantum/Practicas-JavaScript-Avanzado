# 🎯 Adivina el número

Juego interactivo donde el jugador debe adivinar un número secreto entre 1 y 100, desarrollado con **Vite** y **Vanilla JavaScript**.

## 🎮 Características

- ✅ Generación aleatoria de número secreto
- ✅ Pistas visuales ("más alto" / "más bajo")
- ✅ Contador de intentos
- ✅ Historial completo de intentos
- ✅ Validación de entrada (solo números 1-100)
- ✅ Diseño moderno y responsivo
- ✅ Soporte para tecla Enter
- ✅ Estadísticas en consola

## 🚀 Tecnologías

- **Vite** - Build tool y servidor de desarrollo
- **Vanilla JavaScript** - Lógica del juego
- **CSS3** - Estilos modernos con animaciones

## 📋 Requisitos

- Node.js (v16 o superior)
- npm (v7 o superior)

## 🔧 Instalación

```bash
# Clonar el repositorio
git clone <tu-repositorio>

# Navegar a la carpeta
cd adivina-el-numero

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
