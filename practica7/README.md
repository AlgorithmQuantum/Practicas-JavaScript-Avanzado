# 🚀 Mi Exploración Espacial

¡Bienvenido, Explorador Espacial! Esta es tu herramienta personal para registrar y compartir información sobre los planetas que descubres en tu viaje por el universo.

## 🪐 Características

- ✅ Registro de planetas con información detallada
- ✅ Visualización de datos en consola
- ✅ Estadísticas de exploración
- ✅ Agregar nuevos planetas de forma interactiva
- ✅ Mensajes divertidos con cowsay
- ✅ Colores y estilos mejorados con chalk

## 📋 Requisitos

- Node.js (v14 o superior)
- npm (v6 o superior)

## 🔧 Instalación

```bash
# Clonar el repositorio
git clone <tu-repositorio>

# Navegar a la carpeta
cd mi-exploracion-espacial

# Instalar dependencias
npm install
